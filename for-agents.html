For AI Agents

Integration guide for autonomous agents accessing Japanese market intelligence.

Quick Start

Add TRANSCODE to your MCP configuration. No build step required.

Claude Desktop / Cursor / Windsurf

{
  "mcpServers": {
    "transcode": {
      "command": "npx",
      "args": ["-y", "transcode-mcp-server"],
      "env": {
        "TRANSCODE_API_URL": "https://japan-intelligence-api.onrender.com",
        "TRANSCODE_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}

Direct API Access

curl -H "X-API-Key: YOUR_API_KEY" \
  https://japan-intelligence-api.onrender.com/api/v1/briefing

Recommended Agent Workflow

For maximum intelligence, call tools in this order:

1. japan_briefing          → Full market overview (start here)
2. company_intelligence    → Deep dive on specific company
3. company_context         → Historical context (why is it this way?)
4. get_disclosures         → Latest corporate actions
5. detect_events           → Macro anomaly check
